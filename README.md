# 🌦️ Weather App

A simple and responsive **Weather App UI** built using **HTML and CSS**.  
This project displays weather information in a clean and modern layout.

## 🚀 Features

- Clean and responsive weather UI
- Displays city name and weather details
- Temperature display section
- Weather condition icons
- Simple and beginner-friendly code
- Mobile responsive design

## 🛠️ Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and responsive layout
